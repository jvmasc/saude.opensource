## O QUE SÃO MÓDULOS?

Módulos são **aprofundamentos especializados** do Protocolo Mestre. Enquanto o Protocolo Mestre fornece a base unificada (Shot dos Campeões), os módulos permitem personalizar e intensificar áreas específicas.

> **Regra de Ouro:** Domine primeiro o [Protocolo Mestre](/protocolo-mestre). Os módulos são para quem quer ir além.

## QUANDO USAR CADA MÓDULO

### M1: Iodo

```
USE SE:
□ Fadiga crônica
□ Sintomas tireoidianos (frio, ganho peso, lentidão)
□ Brain fog persistente
□ Problemas hormonais
□ Nunca suplementou iodo
□ Temperatura basal <36.5°C

MONITORAMENTO: Temperatura basal semanal
PRÉ-REQUISITO: Screening tireoidiano
```

### M2: Remineralização

```
USE SE:
□ Cãibras, espasmos musculares
□ Insônia, dificuldade relaxar
□ Fadiga generalizada
□ Dieta pobre em nutrientes
□ Uso de medicamentos (depletam minerais)

PRÉ-REQUISITO: Nenhum (pode começar imediatamente)
```

### M3: Detox Halogênios

```
USE SE:
□ Iniciando protocolo de iodo
□ Exposição a flúor/bromo/cloro
□ Sintomas de bromismo (acne, brain fog)
□ Água fluoretada

PRÉ-REQUISITO: Usar junto com M1 (Iodo)
```

### M4: Intestino

```
USE SE:
□ Problemas digestivos crônicos
□ Inchaço, gases, constipação/diarreia
□ Alergias alimentares múltiplas
□ Condições autoimunes
□ Brain fog + fadiga

PRÉ-REQUISITO: M2 (cofatores)
```

### M5: Desparasitação

```
USE SE:
□ Fadiga inexplicável
□ Sintomas digestivos crônicos
□ Nunca fez desparasitação
□ Contato com animais/viagens
□ Bruxismo, coceira anal

PRÉ-REQUISITO: Iniciar após Semana 1 do Protocolo Mestre
```

### M6: Dieta Carnívora

```
USE SE:
□ Quer potencializar resultados do protocolo
□ Inflamação sistêmica
□ Condições autoimunes
□ Múltiplas sensibilidades alimentares
□ Perda de peso resistente

PRÉ-REQUISITO: Compromisso de 4-8 semanas
```

### M7: Água Oxigenada

```
USE SE:
□ Muco excessivo
□ Fadiga pós-refeição
□ Candidíase intestinal
□ Durante desparasitação (potencializa)

PRÉ-REQUISITO: Sem gastrite/úlcera. Uso PONTUAL (max 7 dias)
```

### M8: Bicarbonato Extra

```
USE SE:
□ Acidez gástrica frequente
□ Azia pós-refeição
□ Má digestão
□ Durante desparasitação

PRÉ-REQUISITO: Nenhum (pode usar conforme necessidade)
```

### M9: Boro

```
USE SE:
□ Dores articulares (artrite, artrose)
□ Osteoporose ou risco de osteoporose
□ Candidíase recorrente
□ Sintomas de menopausa
□ Dentes fracos/cáries frequentes
□ Calcificação de tecidos moles

PRÉ-REQUISITO: Função renal normal. NÃO usar na gravidez/lactação.
⚠️ Não disponível na UE (regulamentação)
```

### M10: Higienista Moderno

```
USE SE:
□ Quer maximizar vitalidade além da remissão de sintomas
□ Fadiga crônica não resolvida com protocolos bioquímicos (M1-M5)
□ Interesse em otimização completa (longevidade, performance)
□ Quer integrar hábitos ambientais (sol, grounding, jejum, ritmo)
□ Busca estilo de vida completo (não apenas suplementação)

PRÉ-REQUISITO: Screening + Shot dos Campeões ≥7 dias
⚠️ Contraindicações: Jejum (diabéticos insulino-dependentes, gestantes, crianças, transtornos alimentares)
```

### M11: Fáscia

```
USE SE:
□ Dor crônica sem causa aparente
□ Rigidez matinal inexplicável
□ Fibromialgia (dor generalizada + pontos gatilho)
□ Restrições de mobilidade (não por lesão óssea/articular)
□ Recuperação de lesões (otimizar regeneração fascial)
□ Otimização esportiva (eficiência tensegral)
□ Postura degenerativa (padrões fasciais compensatórios)

PRÉ-REQUISITO: Screening + Shot dos Campeões ≥14 dias
⚠️ Contraindicações: Liberação profunda em áreas com trombose/varizes, infecção ativa, osteoporose severa
```

## ORDEM RECOMENDADA

### Para Iniciantes

```
1. Protocolo Mestre (Shot dos Campeões)
   + Medir temperatura basal (linha de base)
2. M2: Remineralização (cofatores)
3. M1: Iodo + M3: Detox Halogênios (juntos)
   + Monitorar temperatura basal semanal
4. M4: Intestino (se necessário)
```

### Para Resultados Rápidos

```
1. Protocolo Mestre (Shot)
   + Temperatura basal de linha de base
2. M6: Dieta Carnívora
3. M5: Desparasitação
4. M1-M4 conforme necessidade
```

### Para Condições Autoimunes

```
1. Protocolo Mestre
2. M6: Dieta Carnívora (eliminação)
3. M4: Intestino (protocolo 4R)
4. M2: Remineralização
5. M1: Iodo (com preparação Hashimoto se aplicável)
   + Monitoramento intensivo de temperatura basal
```

### Para Estilo de Vida Completo (Otimização + Longevidade)

```
1. Protocolo Mestre (Shot dos Campeões)
2. M10: Higienista Moderno (6 pilares diários)
   + Implementação gradual: Sol+Água+Movimento (Sem 1-2)
   + Adicionar Terra+Ritmo (Sem 3-4)
   + Consolidar Alimento Real (Sem 5-8)
3. M6: Dieta Carnívora (opcional — baseline alimentar)
4. M1-M4 conforme necessidade específica
```
