Módulos

Aprofundamento temático. Cada módulo é um deep dive num componente específico do sistema — para quem quer compreender o mecanismo, não só seguir a receita.

Pré-requisito: Os módulos são aprofundamentos do Protocolo Mestre. Recomenda-se dominar o Protocolo Mestre antes de avançar para os módulos individuais.
